========================================================
ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ЗАПУСКУ TELEGRAM-БОТА
========================================================

Эта инструкция поможет вам установить и запустить Telegram-бота даже если вы никогда этого не делали раньше.


== ЧАСТЬ 1: УСТАНОВКА И ЗАПУСК НА WINDOWS ==

1. УСТАНОВКА PYTHON:
   - Скачайте Python версии 3.9.13 по ссылке: https://www.python.org/downloads/release/python-3913/
   - Прокрутите страницу вниз и выберите "Windows installer (64-bit)"
   - Запустите скачанный файл
   - ВАЖНО: Поставьте галочку "Add Python 3.9 to PATH" внизу окна установщика!
   - Нажмите "Install Now" и дождитесь завершения установки

2. ПРОВЕРКА УСТАНОВКИ:
   - Нажмите Win+R на клавиатуре, введите "cmd" и нажмите Enter
   - В открывшейся командной строке введите:
     python --version
   - Вы должны увидеть "Python 3.9.13" или похожий текст

3. СКАЧИВАНИЕ И НАСТРОЙКА БОТА:
   - Создайте папку для бота, например C:\TelegramBot
   - Скопируйте файлы main.py и config.py в эту папку
   - Откройте файл config.py через Блокнот (правый клик -> Открыть с помощью -> Блокнот)
   - Замените "YOUR_BOT_TOKEN_HERE" на токен, полученный от @BotFather в Telegram
   - Замените числа в списке ALLOWED_USERS на ID пользователей, которым нужен доступ
   - Сохраните файл (Ctrl+S)

4. УСТАНОВКА НЕОБХОДИМЫХ БИБЛИОТЕК:
   - Откройте командную строку (Win+R, введите "cmd", нажмите Enter)
   - Введите следующую команду, чтобы перейти в папку с ботом:
     cd C:\TelegramBot
   - Установите библиотеку aiogram с помощью pip:
     pip install aiogram==3.0.0

5. ЗАПУСК БОТА:
   - В командной строке, находясь в папке с ботом, введите:
     python main.py
   - Если всё настроено правильно, вы увидите сообщения о запуске бота
   - Бот будет работать пока запущена командная строка. Чтобы остановить бота, 
     нажмите Ctrl+C в командной строке


== ЧАСТЬ 2: УСТАНОВКА И ЗАПУСК НА LINUX ==

1. УСТАНОВКА PYTHON:
   - Откройте терминал (обычно Ctrl+Alt+T)
   - Выполните следующие команды по очереди:
     sudo apt-get update
     sudo apt-get install python3.9 python3.9-venv python3-pip

2. ПРОВЕРКА УСТАНОВКИ:
   - В терминале введите:
     python3.9 --version
   - Должно отобразиться "Python 3.9.x"

3. СОЗДАНИЕ ПАПКИ И НАСТРОЙКА БОТА:
   - Создайте папку для бота:
     mkdir ~/telegrambot
   - Перейдите в эту папку:
     cd ~/telegrambot
   - Создайте виртуальное окружение:
     python3.9 -m venv venv
   - Активируйте виртуальное окружение:
     source venv/bin/activate

4. УСТАНОВКА НЕОБХОДИМЫХ БИБЛИОТЕК:
   - Находясь в виртуальном окружении (в начале строки должно быть (venv)), выполните:
     pip install aiogram==3.0.0

5. СОЗДАНИЕ ФАЙЛОВ БОТА:
   - В той же папке создайте файл config.py:
     nano config.py
   - Скопируйте содержимое файла config.py из архива
   - Замените "YOUR_BOT_TOKEN_HERE" на токен от @BotFather
   - Замените числа в ALLOWED_USERS на ID нужных пользователей
   - Сохраните файл: Ctrl+O, затем Enter, затем Ctrl+X
   - Создайте файл main.py:
     nano main.py
   - Скопируйте содержимое файла main.py из архива
   - Сохраните: Ctrl+O, затем Enter, затем Ctrl+X

6. ЗАПУСК БОТА:
   - Убедитесь, что виртуальное окружение активировано (в начале строки должно быть (venv))
   - Запустите бота:
     python main.py
   - Бот будет работать пока запущен терминал. Для остановки нажмите Ctrl+C


== ЧАСТЬ 3: КАК ПОЛУЧИТЬ ТОКЕН И ID ПОЛЬЗОВАТЕЛЕЙ ==

1. ПОЛУЧЕНИЕ ТОКЕНА БОТА:
   - В Telegram найдите @BotFather
   - Напишите ему /newbot
   - Следуйте инструкциям, укажите имя для бота
   - После создания бота, @BotFather пришлет вам токен вида:
     123456789:ABCdefGHIjklMNOpqrSTUvwxYZ
   - Скопируйте этот токен в файл config.py

2. ПОЛУЧЕНИЕ ID ПОЛЬЗОВАТЕЛЯ:
   - В Telegram найдите бота @userinfobot
   - Напишите ему что угодно или /start
   - Он ответит вам сообщением, содержащим ваш ID
   - Этот ID нужно добавить в список ALLOWED_USERS в файле config.py


== ЧАСТЬ 4: УСТРАНЕНИЕ ВОЗМОЖНЫХ ПРОБЛЕМ ==

1. "Python не найден" или "pip не найден":
   - Windows: Убедитесь, что поставили галочку "Add Python to PATH" при установке
   - Linux: Попробуйте использовать python3 и pip3 вместо python и pip

2. "ModuleNotFoundError: No module named 'aiogram'":
   - Убедитесь, что правильно установили библиотеку aiogram
   - Windows: pip install aiogram==3.0.0
   - Linux: pip install aiogram==3.0.0 (в активированном виртуальном окружении)

3. "Бот не отвечает в Telegram":
   - Проверьте, правильно ли указан токен в config.py
   - Убедитесь, что бот запущен (в командной строке/терминале должны быть логи)
   - Если вы только что создали бота, напишите ему /start

4. "Нет доступа":
   - Убедитесь, что ваш ID добавлен в список ALLOWED_USERS в файле config.py
   - Перезапустите бота после изменения списка пользователей

5. "PermissionError" при запуске:
   - Windows: Запустите командную строку от имени администратора
   - Linux: Проверьте права доступа к папке, используйте sudo если нужно

Если у вас возникли другие проблемы, которые не описаны здесь, попробуйте поискать решение 
в интернете, указывая текст ошибки из терминала/командной строки.
